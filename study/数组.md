##js数组的方法

####1.concat

```javascript
var a = [1,2,3];
var b = [4,5,6];

var c = a.concat(b);     
c //  [1,2,3,4,5,6]


理解：  返回一个新的数组c
```

####2.push

```javascript
var a = [1,2,3];
var b = [4,5,6];

var c = a.push(8,9);
a   //1,2,3,8,9
c   //  length长度      5

理解： 直接向a里面推入数组，返回的c是a的长度，注意push里面一定要是直接的数组数据，不能是b，修改a的数据


```

####3.join

```JavaScript
var a = [1,2,3,4,5]; 
var b = a.join("|");
a   //[1,2,3,4,5]
b   //"1|2|3|4|5"


理解：  返回的是字符串，可以用这个将数组转换成字符串
```
