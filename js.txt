前端用rem来自适应开发移动端页面

设计师给的psd图片应设置成750px  或者  640px    fz设置成100(也方便计算)

var util={
    init:function(){
        this.calcREM();
    },
    calcREM:function(){
        var size = 750,
            fz = 100,
            baseDom = document.getElementsByTagName("html")[0],
            calcSz =  baseDom.getBoundingClientRect().width/ size,
            fullSz = calcSz * fz;
        baseDom.style.fontSize = fullSz + "px";
    }
};

util.init();

这个js能在html上生成基础(动态)尺寸font-size，以此页面来根据写的rem值进行自适应
